### Getting Started

To begin, you'll need an API key:

1. Log in to the [Heyy App](https://app.heyy.io).
2. Navigate to [Settings > API Keys](https://app.heyy.io/settings/api-keys).
3. Click the **Create API Key** button.
4. Then copy the newly created API keys and store it somewhere sage

<Note>
For security reasons the key will only appear once and then not be visible anymore.
</Note>

With your API key, you're all set to start using our API.
Happy coding!
