Organize and categorize your contacts using custom labels.

<Frame>
  ![Labels Settings](/images/settings-labels.png)
</Frame>

<Info>
  Go to [**Settings \> Labels**](https://app.heyy.io/settings/labels)
</Info>

## Managing Labels

<Steps>
  <Step title="Create Label">
    Click **\+ Create label** and enter a label name
  </Step>
  <Step title="Search Labels">
    Use the search bar to find existing labels
  </Step>
  <Step title="Edit/Delete">
    Use the action menu to modify or remove labels
  </Step>
</Steps>

## Using Labels

Labels can be applied to contacts to:

- Segment your audience
- Filter conversations
- Create targeted campaigns
- Organize leads by status (e.g., "new lead", "qualified", "customer")

<Tip>
  Create labels for your sales pipeline stages to track leads through your conversion funnel.
</Tip>
