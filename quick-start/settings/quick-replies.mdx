Create saved responses for frequently asked questions to speed up your team's response time.

<Frame>
  ![Quick Replies Settings](/images/settings-quick-replies.png)
</Frame>

<Info>
  Go to [**Settings \> Quick Replies**](https://app.heyy.io/settings/quick-replies)
</Info>

## Creating Quick Replies

<Steps>
  <Step title="Click Create Quick Reply">
    Click **\+ Create quick reply** button
  </Step>
  <Step title="Set Shortcut">
    Create a memorable shortcut (e.g., "/greeting", "/hours")
  </Step>
  <Step title="Write Response">
    Enter the full message content
  </Step>
  <Step title="Save">
    Save your quick reply
  </Step>
</Steps>

## Using Quick Replies

In any conversation:

1. Type "/" to see available quick replies
2. Select the desired reply or type the shortcut
3. The full message will be inserted into your response

<Tip>
  Create quick replies for:

  - Common greetings
  - Business hours
  - Pricing information
  - Return policies
  - FAQ answers
</Tip>
