## Attribute Properties

Each attribute has:

- **Name** - Friendly display name for the attribute.
- **ID** - Unique identifier used in integrations and templates (Can't be edited once added. Only letters, numbers, and underscores are allowed for this field).
- **Description** - Optional description of the attribute.
- **Type** - Data type (Text, Number, Single-select, Date, Date + Time).

<AccordionGroup>
  <Accordion title="Text">

  </Accordion>

  <Accordion title="Number">

  </Accordion>

  <Accordion title="Single-select">

  </Accordion>

  <Accordion title="Date">

  </Accordion>

  <Accordion title="Date + Time">

  </Accordion>

## Attribute Types

<Tabs>
  <Tab title="Text">
    Free-form text input for names, descriptions, notes, etc.
  </Tab>
  <Tab title="Number">
    Numeric values for quantities, scores, amounts, etc.
  </Tab>
  <Tab title="Date">
    Date values for birthdays, signup dates, appointments, etc.
  </Tab>
  <Tab title="Boolean">
    Yes/No values for opt-ins, preferences, status flags, etc.
  </Tab>
</Tabs>

<Note>
  Attributes can be used in message templates with placeholders like `{{attribute_id}}` for personalized messaging.
</Note>
