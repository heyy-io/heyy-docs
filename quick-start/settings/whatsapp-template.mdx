# WhatsApp Templates

Create and manage WhatsApp message templates for outbound communications.

<Frame>
  ![WhatsApp Templates Settings](/images/settings-whatsapp-templates.png)
</Frame>

<Info>
  Go to [**Settings \> WhatsApp Message Templates**](https://app.heyy.io/settings/whatsapp-templates)
</Info>

## Template Overview

WhatsApp requires pre-approved templates for initiating conversations. Your templates table shows:

- **Name** - Template identifier
- **Channel** - Associated WhatsApp channel
- **Status** - Approval status (Active, Pending, Rejected)
- **Category** - Template type (Marketing, Utility, Authentication)
- **Created At** - Creation date

## Creating Templates

<Steps>
  <Step title="Click New Template">
    Click **\+ New Template** button
  </Step>
  <Step title="Configure Template">
    - Enter a template name
    - Select the category
    - Choose the language
    - Write your message content with placeholders
  </Step>
  <Step title="Add Components">
    Optionally add:

    - Header (text, image, video, document)
    - Body with variables
    - Footer
    - Quick reply or call-to-action buttons
  </Step>
  <Step title="Submit for Approval">
    Submit the template to WhatsApp for review
  </Step>
</Steps>

<Warning>
  Templates must comply with WhatsApp's policies. Marketing templates require opt-in consent from recipients.
</Warning>

## Template Categories

<Tabs>
  <Tab title="Marketing">
    Promotional messages, offers, and announcements. Requires explicit opt-in from recipients.
  </Tab>
  <Tab title="Utility">
    Transactional messages like order updates, appointment reminders, and account notifications.
  </Tab>
  <Tab title="Authentication">
    One-time passwords and verification codes.
  </Tab>
</Tabs>
